# Hands-On 6 – SQLAlchemy ORM

## Objective
Implement ORM mapping using SQLAlchemy and perform CRUD operations.

## Files
- models.py
- crud.py
- joinedload.py
- requirements.txt

## Concepts Covered
- SQLAlchemy ORM
- Declarative Base
- Relationships
- CRUD Operations
- Session Management
- N+1 Query Problem
- joinedload()

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Update your MySQL password inside `models.py`

3. Run

python models.py

python crud.py

python joinedload.py

## Outcome

- Tables created using ORM
- CRUD operations performed
- N+1 query problem demonstrated
- Optimized using `joinedload()`
