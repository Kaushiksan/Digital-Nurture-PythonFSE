# Hands-On 1 – Django Project Setup

## Objective

Understand the fundamentals of Django, including project creation, request-response cycle, MVT architecture, middleware, WSGI/ASGI, and URL routing.

## Files

- requirements.txt
- commands.txt
- notes.py
- settings_comments.txt
- views.py
- urls.py

## Topics Covered

- Django Installation
- Django Project Creation
- Django App Creation
- Request–Response Cycle
- MVT Architecture
- Middleware
- WSGI vs ASGI
- URL Routing
- Views

## Commands

Install Django

pip install django

Create Project

django-admin startproject coursemanager

Create App

python manage.py startapp courses

Run Server

python manage.py runserver

## Expected Output

Opening the home page displays:

Course Management API is running

Status: Completed
