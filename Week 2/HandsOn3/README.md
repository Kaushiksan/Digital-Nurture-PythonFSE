# Hands-On 3 – Django Forms, Templates & Generic Views

## Objective

Learn how to build forms, render templates, and use Django Generic Class-Based Views.

## Files

- forms.py
- views.py
- urls.py
- course_list.html
- course_form.html
- requirements.txt

## Topics Covered

- Django Forms
- ModelForm
- Templates
- Generic Views
- ListView
- CreateView
- URL Routing
- CSRF Protection

## Commands

```bash
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
```

## Expected Outcome

- Display list of courses
- Add new course using a form
- Redirect to course list after saving
